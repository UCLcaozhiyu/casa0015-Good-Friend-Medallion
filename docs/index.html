Project Vision

"The Witcher Medallion ‚Äì When danger is near, it trembles..."

Inspired by the Witcher's Medallion

"Find your target, no server required."

Just like the Witcher's medallion vibrates in the presence of magic, this app helps you sense your friends in the real world. The closer you get to your target, the stronger the vibration feedback becomes.

Version History

Version 1.0 - Initial Release

The foundation of our vision - Basic BLE and GPS integration

  • Initial Flutter project setup with core dependencies
  • Basic location services implementation
  • Preliminary BLE scanning functionality
  • Simple vibration feedback system

Version 2.0 - Enhanced Features

Improved functionality and user experience

  • Enhanced BLE scanning with device name filtering
  • Improved location tracking accuracy
  • Dynamic vibration patterns based on proximity
  • Optimized battery consumption

Version 3.0 - Latest Release

A complete transformation with advanced features

  • Sophisticated BLE RSSI smoothing algorithm
  • Enhanced UI with real-time distance indicators
  • Dynamic compass interface pointing to friends
  • Improved device name display and filtering
  • QR code docking functionality
